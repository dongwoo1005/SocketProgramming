# Socket Programming
      
## How to compile and make the programs
`make all`
`make server`
`make client`

## How to run server program
- Number of parameters: 1
- Parameter:
      $1: <req_code>
`./server $1`

## How to run client program
- Number of parameters: 4
- Parameter:
      $1: <server_address>
      $2: <n_port>
      $3: <req_code>
      $4: <message>
`./client $1 $2 $3 $4`
