# Socket Programming
      
## How to compile and make the programs
`make all`
`make server`
`make client`

## server program
~~~~
Number of parameters: 1
Parameter:
      $1: <req_code>
How to run:
      ./server $1
~~~~

## client program
~~~~
Number of parameters: 4
Parameter:
      $1: <server_address>
      $2: <n_port>
      $3: <req_code>
      $4: <message>
How to run:
      ./client $1 $2 $3 $4
~~~~
